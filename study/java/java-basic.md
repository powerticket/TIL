### Primitive Type

| 구분   | Type    | Bit 수 | Value           |
| ------ | ------- | ------ | --------------- |
| 논리형 | boolean | 1      | true / false    |
| 정수형 | byte    | 8      | -128 ~ 127      |
| 정수형 | short   | 16     | -32768 ~ 32767  |
| 정수형 | int     | 32     | -2^31 ~ 2^31-1  |
| 정수형 | long    | 64     | -2^63 ~ 2^63-1  |
| 실수형 | float   | 32     |                 |
| 실수형 | double  | 64     |                 |
| 문자형 | char    | 16     | \u0000 ~ \uffff |
